# express-typescript-APIREST-starter
API REST in Express 4 and TypeScript with the controllers as the JS Class - skeleton

Inspired by:
 - https://github.com/Microsoft/TypeScript-Node-Starter.git

## Fast run

Follow steps:
- yarn 
- yarn run build
- yran run serve
- open http://localhost:3000/
